Greenfield University's Cloud-Powered Library System

Overview

The Instant Library is a virtual library platform developed by Greenfield University's Cloud Solutions Department. It addresses the shortage of physical textbooks faced by the BSC Computer Science department due to a growing student population.

Using Flask for backend development, AWS EC2 for hosting, and DynamoDB for managing data, the system allows students to register, log in, and request books online. AWS SNS sends real-time notifications to students and library staff about requests, ensuring effective communication and resource management.
